42. Print numbers with their squares and cubes for n lines

Write a C program to print a number, its square and cube, starting with 1 and printing n lines. Accept the number of lines (n, integer) from the user.
Test Data :
Input number of lines: 5
Expected Output:
1 1 1
2 4 8
3 9 27
4 16 64
5 25 125

```c
#include <stdio.h>
int main() {
    int a, i, j = 1, x = 0;

    // Prompt for user input
    printf("Input number of lines: ");
    scanf("%d", &a);

    // Loop for each line
    for(i = 1; i <= a; i++) {
        // Print i, i squared, and i cubed
        printf("%d %d %d\n", i, i*i, i*i*i);
    }

    return 0;
}
```
